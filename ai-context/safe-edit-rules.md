# SAFE EDIT RULES

Codex must follow these rules while editing the project.

DO NOT MODIFY:

- animations
- GSAP timelines
- layout structure
- component props
- styling
- Three.js scenes
- canvas logic

ONLY MODIFY:

- text strings
- numbers
- service descriptions
- statistics
- company data

If a section requires structural changes:
do not implement them.

Instead mark:

// STRUCTURE CHANGE REQUIRED
